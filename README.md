## 単回帰させるやつ
made by Ryotaro Sumi  
  
Predict calorie from some value  
  
## How to Use
`/bin/kadai3 food.csv [flag]`  

### flag
C : Carbon  
F : Fat  
G : GI  
P : Protain  
  
## compile
  
### make
`> make` compile difference file  
  
### compile all
  
`> make all` compile all files
